问答小贴士：

1.我该如何注册微信服务号？

通过电脑打开微信公众平台官方网站：http://mp.weixin.qq.com/，点击右上角的“立即注册”...查看详情>>

2.我已经有微信公众号，但是不知道是服务号还是订阅号，该怎么查询？

通过电脑登录微信公众平台官方网站：http://mp.weixin.qq.com/，右上角可查看公众号类型...查看详情>>

3.已有微信订阅号，该如何更换为服务号？

通过电脑登录微信公众平台官方网站：http://mp.weixin.qq.com/，进入公众号设置...查看详情>>
